# 标题表示 #
用井号#来标识，井号与标题之间需要加括号,共有6个等级
# One #
## Two ##
### Three ###
#### Four ####
##### Five #####
###### Six ######

# 换行 #
换行时加\<br>就可以
<br>这样子<br>就实现了换行

但是好像

回车也行


与上一行文本有一行以上的空行就可以起到换行作用


# 强调 #
<br>\*强调*   *斜体*
<br>\*\*加重强调**  **粗体**
<br>\*\*\*特别强调***  ***粗斜体***
   
<br>\_强调_    _斜体_
<br>\_\_加重强调__   __粗体__
<br>\_\_\_特别强调___  ___粗斜体___
